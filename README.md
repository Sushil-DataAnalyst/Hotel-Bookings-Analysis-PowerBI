# Hotel-Bookings-Analysis-PowerBI
An end-to end data analysis project using SQL and Power BI to analysis hotel booking trends, cancellation patterns, and revenue insights.
# 🏨 Hotel Booking Business Intelligence Analysis

## 📌 Project Overview
Is project ka main objective hotel booking data ko analyze karke business insights nikalna hai. Maine isme **SQL** aur **Power BI** ka use karke cancellations, revenue trends, aur customer behavior par focus kiya hai taaki hotel management apne revenue loss ko kam kar sake.

![Hotel Booking Dashboard](Hotel%20Bookings.png)

## 🛠️ Tech Stack Used
* **Database:** SQL (Data Cleaning aur Extraction ke liye)
* **Visualization Tool:** Power BI (Interactive Dashboards ke liye)
* **Data Processing:** Power Query (Data Transformation aur Cleaning ke liye)

## 🔍 Key Business Questions Addressed
* **Cancellation Analysis:** Kaunse factors ki wajah se bookings sabse zyada cancel ho rahi hain?
* **Revenue Insights:** 'Average Daily Rate' (ADR) ka trend mahine-dar-mahine kaisa hai?
* **Customer Profiling:** Kaunse type ke agents ya distribution channels sabse zyada business la rahe hain?

## 💡 Top Insights & Recommendations
* **Lead Time Factor:** Analysis mein pata chala ki jo bookings 60 din pehle hoti hain, unke cancel hone ke chances 30% zyada hote hain.
* **Deposit Policy:** 'No Deposit' wali bookings ka cancellation rate 'Non-refundable' se kaafi zyada hai. **Recommendation:** High-demand seasons mein pre-payment mandatory honi chahiye.
* **Seasonality:** August aur July mein sabse zyada demand rehti hai, wahan dynamic pricing strategy use ki ja sakti hai.

## 📊 DAX Measures & SQL Logic
Maine isme advanced analysis ke liye ye logic use kiye hain:
* **SQL:** Window Functions aur CTEs ka use karke MoM (Month-over-Month) growth calculate ki.
* **DAX:** `Total Revenue`, `Cancellation %`, aur `Average Lead Time` jaise complex measures create kiye.
